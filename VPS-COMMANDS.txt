--------------------------------------------------------------------------------
5. STUCK / FAILED — WHAT TO DO
--------------------------------------------------------------------------------

# A) Poll stuck at N/35, status still "running", no new images for 15–20+ min
#    → zombie or Flow throttle. Check status + logs first:
bash /opt/niche/scripts/vps-run-status.sh RUN_ID
sudo journalctl -u niche-image-worker -n 80 --no-pager

#    If status stuck "running" with old updated_at → restart worker, then resume:
sudo systemctl restart niche-image-worker
sleep 3
sudo bash /opt/niche/scripts/vps-resume-run.sh RUN_ID

# B) status=failed, error has 401 / 502 / Unauthorized
#    → Flow login expired
#    1) SSH+VNC tunnel, sign in at labs.google/fx/tools/flow
#    2) bash /opt/niche/scripts/vps-preflight.sh   (must pass)
#    3) sudo bash /opt/niche/scripts/vps-resume-run.sh RUN_ID

# C) status=failed or preflight generate-image 403 (UNUSUAL_ACTIVITY / reCAPTCHA)
#    → Flow rate limit on current IP. Wait 1–2h, OR switch Chrome to VPN proxy:
sudo bash /opt/niche/scripts/vps-xray-proxy.sh status
sudo bash /opt/niche/scripts/vps-xray-proxy.sh start
sudo bash /opt/niche/scripts/vps-chrome-network.sh proxy
bash /opt/niche/scripts/vps-preflight.sh
sudo bash /opt/niche/scripts/vps-resume-run.sh RUN_ID

# D) Chrome network modes (proxy = xray VPN, direct = VPS public IP)
sudo bash /opt/niche/scripts/vps-chrome-network.sh status
sudo bash /opt/niche/scripts/vps-chrome-network.sh proxy
sudo bash /opt/niche/scripts/vps-chrome-network.sh direct
cat /opt/niche/chrome.env

# E) FlowKit / extension not connected
bash /opt/niche/scripts/restart_flowkit_stack.sh
# or on VNC desktop:
start-chrome-flowkit
# then open Flow tab and confirm extension connected; re-run preflight

# F) Poll timed out but VPS later finished (status=complete)
#    → Just re-run GHA wait-for-images; images are already on disk

# G) Poll 0 images for 15+ min
#    → Job never started or FlowKit dead. Preflight → resume (same RUN_ID)

--------------------------------------------------------------------------------
12. COMMON ERROR → FIX
--------------------------------------------------------------------------------

401 / 502 on generate-image or create-project
  → VNC Flow re-login → vps-preflight.sh → resume same RUN_ID

403 UNUSUAL_ACTIVITY / reCAPTCHA
  → Wait 1–2h OR vps-chrome-network.sh proxy → preflight → resume

429 rate limit
  → Wait; resume same RUN_ID (FLOW_RESUME_COOLDOWN_SEC=600 on resume)

Poll timeout while still progressing slowly
  → Often Fine; check VPS status later; if complete, re-run wait-for-images

Zombie status=running, no progress
  → systemctl restart niche-image-worker → vps-resume-run.sh RUN_ID
